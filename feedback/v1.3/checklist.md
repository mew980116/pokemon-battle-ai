# V1.3 实测 Checklist

> 用 `mew's` 账号对战，PO 脚本控制台收集 print_s 日志。
> v1.3：改动 I(被迫换人纳入高standard) + softmax 候选质量(M溢出折减+命中率 / K先制建模 / J上下位剔除 / L自损非首选)。
> 逻辑见 [docs/v1.3-battle-logic.md](../../docs/v1.3-battle-logic.md)。

---

## A. 改动 I：被迫/KO 换人选对目标

- [ ] **case7-R2 复盘**：坚果哑铃被秒后，standard 最高的超坏星(+35)被换上(或并入候选竞争)，不再换 standard 最低的三首恶龙(-65)
- [ ] **case4-R8 复盘**：standard 最高的千面避役(+5)被选，不再换 standard -35 的人造细胞卵
- [ ] 被迫换人日志出现 `改动I 被迫换人合并候选:[...] bestStd:N`，候选含高 standard 的墙
- [ ] **case6-R7 复盘(主动换人不变)**：主动换人仍选 standard 最高的戽斗尖梭，改动I 未影响主动路径
- [ ] 被迫换人未变得过度保守(墙输出为0时仍能在攻防间合理取舍)

## B. 改动 M：溢出折减 + 命中率

- [ ] **case4-R12 复盘**：水炮(命中80,伤害158%)与冲浪(命中100,伤害130%)都能秒时，选冲浪
- [ ] 溢出招(伤害远超对手当前HP)的 softmax 权重不再因名义高伤害而虚高
- [ ] 高命中招在伤害接近时权重高于低命中招
- [ ] 残血对手(foeHp低)时，刚好能秒的招不被误判为大幅溢出(1.10阈值保护)
- [ ] 验证不影响斩杀：能斩杀的招仍走 commandDecided 确定性路径(不受溢出折减影响)

## C. 改动 K：先制度建模

- [ ] **case7-R3 复盘**：霸道熊猫对皮可西，子弹拳(先制+1,可斩杀,对手能秒我)被选中，不再抽到拍落被秒
- [ ] 我方稳定更快(速度上界都低于我方)时，先制招不被额外加权(无谓加权)
- [ ] 我方可能更慢/稳定更慢 + 先制可斩杀时，先制招权重显著提升
- [ ] 突袭(296)在我方稳定更快且有更高斩杀招时被降权(×0.3)，但仍保留小概率
- [ ] 突袭在残血对手/拼血场景(无更高斩杀招)时正常参与

## D. 改动 J：上下位招剔除

- [ ] 同属性冗余招(如冷冻干燥/冰冻光束、十万伏特/电击)中，下位招日志出现 `改动J 剔除下位招:X(被Y支配)`
- [ ] 被剔除的下位招不再进入 softmax 候选(candidates 减少)
- [ ] 先制不同的同属性招(子弹拳 vs 拍落)不被错误互相支配
- [ ] 命中不同的同属性招(冲浪 vs 水炮)，低命中招被高命中招支配(依赖M先生效)

## E. 改动 L：自损 debuff 招非首选剔除

- [ ] **case7 流星群复盘**：流星群非首选(有更高伤害的非自损招)时被剔除 `改动L 剔除非首选自损招:流星群`
- [ ] 流星群能斩杀(最高分)时保留，正常参与
- [ ] 过热/精神突破/青草混合/花朵加农炮等同类自损招行为一致

## F. 不退化回归(重点)

- [ ] softmax 候选数合理，未退化为纯贪心(candidates 经常 >1)
- [ ] 未出现候选清空(若清空保险触发，日志应能反映)
- [ ] 同属性多招但伤害/命中/先制有真实差异时，不被过度剔除
- [ ] 换人频率(改动I)与 v1.2 量级相当，未过度换人
- [ ] 整体对战胜率/手感不低于 v1.2
