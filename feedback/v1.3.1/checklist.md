## A. 改动1：KO 换人选对目标（最优先，致命 bug 修复）

- [ ] **case1-R6 复盘**：KO 后换人选 standard 最高的（勾帕路翁35），不再选 standard 5 的达摩狒狒
- [ ] **case2-R6 复盘**：KO 后换人选 standard 最高的（盖欧卡95），不再选 standard -75 的铁螯龙虾送死
- [ ] KO 换人日志出现 `改动I 被迫换人纯standard候选:[...]`（说明走了 passive 路径）
- [ ] KO 换人不再出现"选 standard 最低攻击候选送死"

## B. 改动2：被动换人纯 standard 候选

- [ ] **case2-R0 复盘**：被动换人候选不含 standard=-130 的铁螯龙虾（去掉攻击候选无条件纳入）
- [ ] 被动换人候选都是 standard 接近最高(容差30)且非被克制(>-50)的
- [ ] 被动换人日志 `改动I 被迫换人纯standard候选:` 候选列表合理

## C. 改动3：standard 虚高缓解

- [ ] **丰蜜龙复盘**：丰蜜龙对爱管侍(被妖精2x克制+输出低) standard 不再虚高到 50，应明显回落
- [ ] 输出过低的墙（maxmovepow/满血<0.18）standard 被罚 -30
- [ ] 改动3① L1752 死代码修复后，危险属性(岩/妖/鬼/火/电/冰)额外 -20 生效，整体 standard 未异常偏移
- [ ] 正常高耐久墙（抗得住+有输出）standard 不被误罚

## D. 改动4：回退 J，保留 L

- [ ] **case1-R13 复盘**：伏特替换不再被误删，softmax 候选含伏特替换
- [ ] 改动 L（自损非首选剔除）仍正常工作：流星群等自损招非首选时仍被剔除
- [ ] softmax 候选数合理，未因回退 J 而异常增多/减少

## E. 不退化回归

- [ ] 换人频率与 v1.3 量级相当，未过度换人
- [ ] standard 评分整体合理（改动3①激活-20后未导致大面积偏低）
- [ ] softmax 决策质量不低于 v1.3（改动M/K 仍正常）
- [ ] 整体对战胜率/手感不低于 v1.3
